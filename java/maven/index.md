# Maven标准生命周期
个完整的项目构建过程通常包括清理、编译、测试、打包、集成测试、验证、部署等步骤，Maven从中抽取了一套完善的、易扩展的生命周期。Maven的生命周期是抽象的，其中的具体任务都交由插件来完成。Maven为大多数构建任务编写并绑定了默认的插件
1、clean：项目清理的处理
2、default(或 build)：项目部署的处理
3、site：项目站点文档创建的处理
