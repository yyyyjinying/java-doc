## npm
- npm view node versions
- nrm ls
- nvm install 8.14.0 // 安装v8.14.0版本的node.js。
- nvm use 8.14.0
- nrm use npm
- npm login
- npm publish
  
## n 安装和切换node版本
- sudo npm i -g n
- n -V
- n ls
- n xx.xx.x (xx.xx.x 为要安装的版本号)
- n lastest
- n stable
- n rm xx.xx.x
- n use xx.xx.x
- n use xx.xx.x a.js
